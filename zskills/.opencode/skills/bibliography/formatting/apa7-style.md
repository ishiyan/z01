# APA 7th Edition Style

Default citation style for this skill. Based on the *Publication Manual of the American Psychological Association*, 7th edition (2020).

## In-Text Citations

### Basic Forms

- **One author**: (Smith, 2023)
- **Two authors**: (Smith & Jones, 2023)
- **Three or more**: (Smith et al., 2023)
- **Narrative**: Smith (2023) found that...
- **Multiple sources**: (Brown, 2021; Smith et al., 2023; Zhang, 2024) — alphabetical order

### Page Numbers

Include for direct quotes or specific passages:
- (Smith, 2023, p. 45)
- (Smith, 2023, pp. 45–47)

### Same Author, Same Year

Disambiguate with lowercase letters: (Smith, 2023a), (Smith, 2023b)

## Reference List Entries

### Journal Articles

**Format**:
```
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article in sentence case. *Journal Name in Title Case*, *volume*(issue), page–page. https://doi.org/xx.xxx/yyyy
```

**Examples**:
```
Smith, J. D., Johnson, M. L., & Williams, K. R. (2023). Machine learning approaches in drug discovery. *Nature Reviews Drug Discovery*, *22*(4), 301–318. https://doi.org/10.1038/nrd.2023.001

Tillson, T. (1998). A better moving average. *Technical Analysis of Stocks and Commodities*, *16*(1), 33–37.
```

**21+ authors**: List first 19, then ... then last author.

### Books

**Format**:
```
Author, A. A. (Year). *Title of work: Subtitle in sentence case* (edition). Publisher. https://doi.org/xxxx
```

**Examples**:
```
Kumar, V., Abbas, A. K., & Aster, J. C. (2021). *Robbins and Cotran pathologic basis of disease* (10th ed.). Elsevier.

Oppenheim, A. V., & Schafer, R. W. (2010). *Discrete-time signal processing* (3rd ed.). Pearson. Available at: https://books.google.com/books?id=EaMgAwAAQBAJ
```

**Note**: The Google Books URL is an optional extension (not standard APA). Include when the user requests it or when writing for web-published technical articles.

### Edited Book Chapters

**Format**:
```
Author, A. A. (Year). Title of chapter. In E. E. Editor (Ed.), *Title of book* (pp. xx–xx). Publisher. https://doi.org/xxxx
```

### Conference Papers

**Format**:
```
Author, A. A. (Year, Month day–day). Title of paper. In E. Editor (Chair), *Title of symposium* [Conference presentation]. Conference Name, Location.
```

Or for published proceedings:
```
Author, A. A. (Year). Title of paper. In E. Editor (Ed.), *Proceedings of conference name* (pp. xx–xx). Publisher.
```

### Preprints

**Format**:
```
Author, A. A., & Author, B. B. (Year). Title of preprint. *Repository Name*. https://doi.org/xxxx
```

**Example**:
```
Zhang, Y., & Chen, L. (2024). Novel therapeutic targets. *bioRxiv*. https://doi.org/10.1101/2024.01.001
```

### Websites and Online Sources

**Format**:
```
Author, A. A. (Year, Month Day). *Title of page*. Site Name. https://www.example.com/page. Last accessed: 2025-04-30.
```

**Rules**:
- Include "Last accessed: YYYY-MM-DD" for URLs without DOIs
- If author and site name are the same, omit site name
- If no date: use (n.d.)
- If no author: start with title

**Example**:
```
Jurik, M. (1999). *How it works*. Jurik Research. http://jurikres.com/faq1/faq_ama.htm#how_work. Last accessed: 2025-04-12.
```

### Software and Code

**Format**:
```
Author, A. A. (Year). *Software name* (Version X.X) [Computer software]. Publisher/Repository. https://url
```

### Forum Posts

**Format**:
```
Author, A. A. [username]. (Year, Month Day). Title of post [Online forum post]. Forum Name. https://url. Last accessed: YYYY-MM-DD.
```

## Formatting Rules

### Author Names
- Last name, First initial. Middle initial.
- Separate multiple authors with commas; use `&` before the last author
- Up to 20 authors: list all
- 21+: list first 19, then `...`, then last author

### Titles
- **Article/chapter titles**: Sentence case (capitalize first word, proper nouns, first word after colon)
- **Journal/book titles**: Title Case, italicized
- **Italicize**: book titles, journal names, volume numbers

### DOIs and URLs
- Always include DOI when available, formatted as: `https://doi.org/xx.xxx/yyyy`
- No period after DOI or URL (it's the last element)
- For URLs without DOIs: include "Last accessed: YYYY-MM-DD"

### Numbers
- Use en-dash (–) for page ranges, not hyphen (-)
- Edition: (2nd ed.), (3rd ed.), (Rev. ed.)

### Ordering
- Alphabetical by first author's last name
- Same author: chronological (oldest first)
- Same author, same year: add lowercase letter suffix (2023a, 2023b)

## Google Books Extension

Not part of standard APA 7 but useful for web-published technical writing.

**When to include**: User requests it, or the article will be published online where readers benefit from direct book access.

**Placement**: After the standard entry, before the period:
```
Oppenheim, A. V., & Schafer, R. W. (2010). *Discrete-time signal processing* (3rd ed.). Pearson. Available at: https://books.google.com/books?id=EaMgAwAAQBAJ
```

**How to find**: Search `https://books.google.com/` for the book title. Use the stable `books?id=XXXXX` URL, not the search result URL.
