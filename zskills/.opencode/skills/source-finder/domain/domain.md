# Domain-Specific Sources

Extensible registry of domain-specific source lists. Add new domain files as needed.

## Available Domains

| Domain | File | Topics covered |
|--------|------|----------------|
| Trading & quantitative finance | [trading.md](trading.md) | Indicators, algorithms, strategies |

## Adding a New Domain

Create a new `.md` file in this folder with:

1. **Source list** — URLs of forums, databases, and communities
2. **Search patterns** — how to search each source effectively
3. **Quality signals** — domain-specific trust indicators
4. **BibTeX notes** — any special citation considerations

The file will be automatically discovered when the topic matches.
