# Substack (Note + Cross-Post Blurb)

Two formats: Substack Note (short teaser) and Cross-Post Blurb (for other newsletter writers to paste).

---

## Substack Note

### Constraints

| Parameter | Value |
|-----------|-------|
| Length | 60-180 words (strict) |
| Voice | Closest to the original essay |
| Closing | Bolded maxim (verbatim from spine) |
| Hashtags | Zero |
| Emoji | Zero |
| Exclamation points | Zero |
| CTA | None (just the link) |

### Structure

```
{hook line — confession preferred}

{body: 2-5 short paragraphs, 1-3 sentences each}

**{closing_maxim from spine — verbatim, bolded}**

Full essay: [{title}]({url})
```

### Rules

1. Under 60 words = teasing. Over 180 words = dwell-time drops. Stay in range.
2. Use 2-3 spine claims with highest translatability scores.
3. Bolded closer is verbatim from `spine.closing_maxim`. Do not edit.
4. If the essay's closer is hedged ("I do not know"), the Note's closer must be hedged too.
5. Single link line at the end. Not mid-text.
6. No hashtags. No emoji. No exclamation points. No custom CTA.
7. Can use one-sentence pivot paragraphs and short reframes.

---

## Cross-Post Blurb

For Substack's cross-post feature. Another newsletter writer pastes this into their cross-post popup.

### Constraints

| Parameter | Value |
|-----------|-------|
| Length | 60-140 words |
| Voice | Third person throughout |
| Tone | Neutral-analytical (curator, not fan) |

### Structure

```
In this piece, [Author] argues {thesis}. {2-3 sentences summarizing the argument + one concrete anchor}. For anyone [building/working with] {domain}, the {specific claim} is the insight that reframes the problem. Read the full piece.
```

### Rules

1. Third person throughout. Never "I argue..." — the blurb will be pasted by another writer.
2. 60-140 words. Substack's cross-post popup is small.
3. Never say "amazing," "brilliant," "must-read." The other writer should sound like a curator, not a fan.
4. No subscriber CTA. Just positioning + "Read the full piece."
5. Preserve paper attributions (Author, Institution, Year).
6. Voice register: neutral-analytical. Not the writer's confessional register.
