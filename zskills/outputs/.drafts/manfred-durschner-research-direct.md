## Key Findings

### Biography
- Name: Dr. Manfred G. Dürschner
- Degree: PhD in Physics ("promovierte Physiker")
- Location: Germany (writes in German, publishes with German publishers, member of German TA association)
- Profession: Active stock market trader + researcher in technical analysis
- Approach: Applies physics/signal processing to financial markets
- Award: VTAD Award 2011, First Prize, for "Gleitende Durchschnitte 3.0"
- No photos found publicly available
- No university affiliation mentioned — appears to be independent researcher/trader

### Publications
1. "Gleitende Durchschnitte 3.0" (Moving Averages 3.0) — VTAD, 2011
2. "Moving Averages 3.0" — IFTA Journal, 2012 Edition, pp. 14-19 (English translation)
3. "Technische Analyse mit EMD" — Wiley-VCH, 2014, ISBN 978-3-527-50718-4
4. "EMD for Technical Analysis" — ResearchGate, ~2019

### Technical Indicators
1. **New Moving Average (NMA) / 3rd Generation Moving Average** — lag-free MA using Nyquist-Shannon theorem
2. **Nyquist-Shannon Moving Average** — specific implementation variant
3. **NWMA Trading System** — Aroon(5) on NWMA(89,21) + Inverse Fisher Transform
4. **StochRSI Intraday System** — StochRSI(3,5) on NWMA(8,3) + Inverse Fisher Transform
5. **EMD-based indicators** — application of Empirical Mode Decomposition to financial time series (described in 2014 book, implementation in Mathematica)
